🏠 Hostel Management System (Frontend)
A lightweight, vanilla JavaScript interface for managing rooms, tenants, and complaints. This frontend interacts with a RESTful API to provide a real-time dashboard and management utility.

🚀 Features
Live Dashboard: Real-time counters for total rooms, active tenants, and pending complaints.

Room Management: Add new rooms with specific rent amounts and view a list of all available rooms.

Tenant Tracking: Register tenants and assign them to specific rooms.

Complaint System: Submit and track maintenance or service issues.

🛠️ Tech Stack
Frontend: HTML5, CSS3, Vanilla JavaScript (ES6+)

API Communication: Fetch API

Backend (Required): Node.js/Express (or any server running on http://localhost:5000)
